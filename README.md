# Gemini Clone

1.  **Install the node packages**
```
npm i
```
 
2.  **Add your api key to gemini.js file**
3.  **Start the server**
```
npm run dev
```

  **To get your own api key** [GEMINI GOOGLE BARD](https://ai.google.dev/)
